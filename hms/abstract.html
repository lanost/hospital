Abstract

Patient portal is an online website developed in HTML ,PHP with the database as MYSQL.The main objective of the project is to assit the patient to register into a hospital and book appoinments.The application consist of a Patient Registery module where the patient can register using a valid phone number.The phone number is validated using an OTP that is sent to the patient’s registered phone number.The patient can book an appointment for the concerned doctor through Appointment module.Once the appointment is booked ,a google invitation is added to the patient’s google calendar.The patient can also view all the appointments.
